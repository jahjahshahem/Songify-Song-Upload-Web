Song CRUD Application

A simple Laravel web application for managing songs, albums, and artists. This project demonstrates basic CRUD functionality, modern UI using Tailwind CSS, and Alpine.js for interactivity.

⸻

Features
	•	List songs with pagination and human-readable timestamps.
	•	Create, edit, and delete songs with validation.
	•	Show details of each song on a dedicated page.
	•	Minimalistic, modern UI using Tailwind CSS (beige/white palette).
	•	Flash messages for success feedback using Alpine.js.
